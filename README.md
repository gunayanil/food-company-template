# Food Delivery

This **fully responsive** website project was developed for food delivery companies.

This template consists of six main sections:

- Features
- How It Works
- Cities (locations)
- Testimonials
- Plans and Prices
- Contact (with a map)

### Technologies and Tools Used
- HTML5
- CSS3
- jQuery (and gmap.js)
